Block-Level and Inline Elements

Every HTML element has a default display value, block and inline. A block-level element starts on a new line and stretches out to the right and left as far as it can.
The inline element does not start on a new line and only takes up as much width as necessary.

Div and Span Elements

The div element can be used as a container for other elements. The span element can be used as a container for text.

HTML Grouping Tags
Tag Description
div Defines a section in a document (block-level)
span Defines a section in a document (inline)
